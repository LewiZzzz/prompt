# prompt

### 改写论文#2

改写论文中的某一部分内容，以使其更加清晰、连贯并且简洁。

```
I'm writing a paper on [topic]. What I tried to say in the following section is [specific point]. Please rephrase it for clarify, coherence, and conciseness, ensuring each paragraph flows into the next. Remove jargon. Use a profession tone.
---
我正在写一篇关于[主题]的论文。我想在以下部分表达的观点是[具体内容]。请帮助我重新措辞，使其更加清晰、连贯且简洁，确保每个段落之间衔接顺畅。去掉复杂术语和行话，保持专业语气。
```

### 改写论文#2

对文本进行改写，以符合学术论文的写作规范

```
Please help me to re-export this text by replacing synonyms, splitting a sentence into two, speaking a word into a phrase, etc., so that the passage is more logically in line with the norms of an academic paper, in xxx words or so. Deletion of repetitive summarizing statements.
---
请帮助我重新输出这段文本，替换同义词，将长句拆分为两个句子，将单词扩展为短语等，使该段落在逻辑上更加符合学术论文的规范，字数控制在大约 xxx 字左右，并删除重复的总结性陈述。
```
